## 📂 Estrutura e Evolução do Repositório
- **Módulos 05 a 07(Fundamentos CLI):** Foco total no fluxo de compilção manual. Exercícios desenvolvidos e executados diretamente via terminal Linux, utilizando `javac` para compilção e `java` para execução dos arquivos `.class`
- **Módulos 10 em diante(Ambiente VS Code):** Localizados dentro da pasta `vscode`. Está pasta contém o projeto estruturado para a IDE, incluindo todos os exercícios práticos e as pastas de laboratórios (`labs`) com as resoluções propostas no curso.
